# SmartCart

## Setup Instructions

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)
- Git

### Installation

1. Clone the repository
```bash
git clone [repository-url]
cd SmartCart
```

2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
The frontend will be available at `http://localhost:5173`

3. Backend Setup
```bash
cd backend
npm install
npm run dev
```
The backend will be available at `http://localhost:3000`
